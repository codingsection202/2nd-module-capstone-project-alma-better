Harsh E-Commerce Web Application

 ##  Project Overview
This project is a fully functional **E-commerce web application** built using modern frontend technologies. It allows users to browse products, view product details, search items, and manage their cart and wishlist.

The project is developed as part of a **Capstone Project** to apply and demonstrate concepts of React, Redux, API integration, and responsive UI design.

---

  ##  Features

  -  **E-commerce Dashboard**
  - Header with logo, search, cart,contact(as wishlist)
  - Banner with animations
  - Product listing from API
  - Footer with company info
  - responsive UI DESIGN

  -  **Search Functionality**
  - Search products by name

  -  **Category Filter**
  - Filter products by category (Electronics, Fashion, etc.)

  -  **Product Details Page**
  - Detailed product information
  - Pricing, reviews, offers, delivery details

  -  **Cart System**
  - Add/remove products
  - Dynamic total price calculation
  - Cart item count display

  - **Login Page**
  - Basic email & password validation

 **Local Storage**
  - Persist cart and wishlist data
